## Week 8 — Research Activity: Programming Language (Lua)

### What is the language used for?
Lua is a lightweight scripting language commonly used for game development, embedded systems, and application scripting. It is often used in game engines such as Roblox and World of Warcraft for scripting game logic.

### Who uses the language?
Lua is used by game developers, embedded system engineers, and companies that need a fast and flexible scripting language. It is especially popular in the gaming industry.

### Useful resources
- https://www.lua.org/docs.html
- https://www.tutorialspoint.com/lua/index.htm

### Why these resources are useful
The official Lua documentation provides accurate and detailed information about the language, while Tutorialspoint offers beginner-friendly explanations and examples that make learning easier.

## Week 8 — User Stories

### General User Story
As a user, I want to create and manage tasks so that I can stay organized and track my work.

### More Specific User Stories
- As a user, I want to add a new task so that I can track work that needs to be completed.
- As a user, I want to delete tasks so that I can remove completed or unnecessary items.
- As a user, I want to filter tasks by priority so that I can focus on important work.
- As a user, I want to move tasks between columns so that I can track progress.

### Acceptance Criteria
- User can create a task using a form
- Task appears in the correct column
- User can delete tasks
- User can filter tasks by priority and status
- Task board updates dynamically when changes are made

## Week 8 — Reflection

This week introduced the concept of user stories and functional requirements. I learned that writing from the user's perspective helps clarify what features are needed in an application.

Researching Lua showed me that different programming languages are designed for specific purposes. I found it interesting that Lua is widely used in game development, which is something I had not considered before.

Writing user stories for my own project helped me better understand the features I need to implement, such as task creation, filtering, and progress tracking.

Overall, this week helped me think more about planning and design rather than just coding.
